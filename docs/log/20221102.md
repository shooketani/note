- ⇒java11をインストールして、以下の通りに変更したところ解消。

```
C:\SpringBoot\tutorial>SET JAVA_HOME=C:\Program Files\Java\jdk-11

C:\SpringBoot\tutorial>set JAVA_HOME
JAVA_HOME=C:\Program Files\Java\jdk-11
```
